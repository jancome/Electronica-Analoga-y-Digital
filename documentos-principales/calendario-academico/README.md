# Calendario académico

En esta carpeta se ubicará el calendario académico oficial del periodo 2026-2 y un resumen de fechas importantes para la asignatura.

## Fechas clave consideradas

- Inicio de clases: 3 de agosto de 2026.
- Semana de receso: 5 al 11 de octubre de 2026.
- Finalización de clases: 30 de noviembre de 2026.
- Cierre académico: diciembre de 2026.

## Uso dentro del curso

El calendario se utilizará para organizar cortes, laboratorios, talleres, entregas y sustentaciones.
