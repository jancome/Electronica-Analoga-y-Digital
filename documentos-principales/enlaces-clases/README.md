# Enlaces de apoyo para clases

En esta carpeta se organizarán videos, simuladores, lecturas y recursos externos útiles para cada unidad del curso.

## Categorías sugeridas

- Introducción a sistemas digitales.
- Sistemas numéricos.
- Aritmética binaria.
- Compuertas lógicas.
- Álgebra de Boole.
- Teorema de De Morgan.
- Mapas de Karnaugh.
- Sumadores y comparadores.
- Codificadores y decodificadores.
- Multiplexores y demultiplexores.
- Flip-flops y contadores.

## Recomendación

Los enlaces deben revisarse antes de compartirlos para validar que sigan disponibles y que el contenido sea apropiado para el nivel del curso.
