# Modelo de informe de laboratorio

En esta carpeta se ubicará el modelo oficial de informe de laboratorio y las instrucciones de entrega.

## Estructura mínima del informe

1. Título de la experiencia.
2. Integrantes.
3. Roles y aportes de cada estudiante.
4. Resumen.
5. Palabras clave.
6. Introducción.
7. Preinforme o trabajo realizado durante la clase.
8. Resultados: circuitos, tablas, mediciones, capturas y fotos.
9. Análisis de resultados.
10. Cálculos solicitados.
11. Conclusiones.
12. Bibliografía.

## Aporte individual

Cada estudiante debe indicar con claridad la parte en la que trabajó. Esta información hace parte de la evaluación de la guía y puede afectar la nota individual.

## Recomendación

El informe debe presentar análisis técnico. No basta con pegar imágenes o capturas de simulación. También se debe evidenciar qué se hizo durante la clase, qué resultados se obtuvieron y cuál fue el aporte de cada integrante.
