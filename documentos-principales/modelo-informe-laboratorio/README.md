# Modelo de informe de laboratorio

En esta carpeta se ubicará el modelo oficial de informe de laboratorio y las instrucciones de entrega.

## Estructura mínima del informe

1. Título de la experiencia.
2. Integrantes.
3. Resumen.
4. Palabras clave.
5. Introducción.
6. Resultados: circuitos, tablas, mediciones, capturas y fotos.
7. Análisis de resultados.
8. Cálculos solicitados.
9. Conclusiones.
10. Bibliografía.

## Recomendación

El informe debe presentar análisis técnico. No basta con pegar imágenes o capturas de simulación.
