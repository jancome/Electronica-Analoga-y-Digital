# Presentaciones del curso

Las presentaciones complementan los marcos teóricos. No sustituyen las guías, ejercicios, simulaciones ni actividades del proyecto ABP.

## Formato

El formato maestro es PowerPoint (`.pptx`) por su compatibilidad con Microsoft Teams, ecuaciones, tablas, circuitos y trabajo institucional.

Cada presentación mantiene:

- Pregunta o situación inicial.
- Conexión con conocimientos previos.
- Resultados de aprendizaje y criterios de éxito.
- Desarrollo visual de conceptos.
- Ejemplo resuelto.
- Ejercicio o discusión activa.
- Error frecuente.
- Relación con laboratorio y ABP.
- Cierre y autoevaluación.
- Fuentes de consulta.

## Accesibilidad

- Texto legible y contraste suficiente.
- El color no es el único recurso para diferenciar información.
- Tablas y diagramas con etiquetas claras.
- Imágenes técnicas verificadas.
- Referencias y términos explicados.

## Uso en Teams

El mismo archivo PPTX puede publicarse como material de clase en Teams. Las diapositivas destinadas al docente pueden llevar notas del presentador, mientras el contenido visible se mantiene dirigido al estudiante.
