# Semana 03 – Transistor BJT y control de cargas

## Propósito

Analizar el BJT como interruptor y diseñar una etapa capaz de activar una carga de baja potencia mediante una señal de control.

## Trabajo de la semana

- Terminales y hoja de datos del BJT.
- Diferencia entre NPN y PNP.
- Corte, región activa y saturación.
- BJT NPN como interruptor de lado bajo.
- Cálculo de corriente y resistencia de base.
- Control de LED, relé o carga DC.
- Diodo de protección para cargas inductivas.
- Potencia y límites del dispositivo.

## Actividad práctica

Lab A02: control de LED y carga de baja potencia. La etapa de amplificación se conserva como actividad de profundización y no desplaza el montaje esencial de conmutación.

## Avance ABP

Cada grupo debe justificar si requiere una etapa BJT, calcularla y explicar cómo se integra con su fuente DC y con la salida que desea controlar.

## Relación con la siguiente semana

La Semana 04 estudia el MOSFET y compara ambos dispositivos para escoger la alternativa más conveniente para el proyecto.
