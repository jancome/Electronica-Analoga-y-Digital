# Semana 08 – Sistemas numéricos

## Objetivo

Realizar conversiones entre sistemas decimal, binario, hexadecimal y BCD para representar información digital.

## Clase de 3 horas

- Sistema decimal.
- Sistema binario.
- Sistema hexadecimal.
- Código BCD.
- Bit, nibble, byte y palabra.
- Ejercicios guiados de conversión.

## Entregable

Taller de conversión entre sistemas numéricos.
