# Semana 08 – Compuertas lógicas

## Objetivo

Analizar el funcionamiento de las compuertas lógicas básicas y universales mediante tablas de verdad, símbolos lógicos y medición de niveles digitales.

- Niveles lógicos 0 y 1.
- Compuertas AND, OR y NOT.
- Compuertas NAND y NOR.
- Compuertas XOR y XNOR.
- Tablas de verdad.
- Circuitos integrados lógicos TTL/CMOS.
- Relación con el Lab 01.
