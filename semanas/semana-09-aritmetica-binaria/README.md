# Semana 09 – Aritmética binaria

## Objetivo

Resolver operaciones aritméticas básicas en sistema binario y analizar su aplicación en circuitos digitales.

## Clase de 3 horas

- Suma binaria.
- Resta binaria.
- Complemento a 1.
- Complemento a 2.
- Números con signo.
- Overflow.
- Ejercicios guiados y trabajo por grupos.

## Entregable

Taller de aritmética binaria.
