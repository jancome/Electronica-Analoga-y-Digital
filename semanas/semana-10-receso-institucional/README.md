# Semana 10 – Receso institucional

## Estado

No se programa clase presencial ni actividad evaluativa nueva durante la semana del **5 al 11 de octubre de 2026**.

## Propósito académico

Esta semana se utiliza como pausa institucional. Para no afectar la secuencia del curso, no se introduce tema nuevo. El estudiante debe aprovecharla para repasar los contenidos del segundo corte vistos antes del receso.

## Recomendación de trabajo autónomo

- Repasar sistemas numéricos.
- Finalizar ejercicios de aritmética binaria.
- Revisar compuertas lógicas.
- Repasar leyes del álgebra booleana.
- Repasar teoremas de De Morgan.
- Preparar dudas para mapas de Karnaugh.

## Entregable

No hay entregable nuevo, salvo actividades pendientes previamente asignadas.
