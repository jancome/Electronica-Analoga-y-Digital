# Semana 11 – Compuertas lógicas

## Objetivo

Reconocer y comprobar el funcionamiento de las compuertas lógicas básicas mediante tablas de verdad, simulación y montaje.

## Clase de 3 horas

- Compuertas AND, OR y NOT.
- Compuertas NAND, NOR, XOR y XNOR.
- Símbolos lógicos.
- Tablas de verdad.
- Expresiones booleanas.
- Desarrollo del Lab 01.
- Evaluación del Corte 2, si aplica según programación.

## Entregable

Tablas de verdad, simulaciones y evidencias del Lab 01.
