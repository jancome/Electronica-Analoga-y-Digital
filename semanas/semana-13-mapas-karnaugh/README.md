# Semana 13 – Comparadores y paridad

## Objetivo

Analizar circuitos combinacionales usados para comparar números binarios y generar o comprobar bits de paridad para detección básica de errores.

## Marco teórico

- [Marco teórico – Semana 13](marco-teorico.md)

## Clase de 3 horas

- Comparador de igualdad.
- Comparador de magnitud.
- Salidas A>B, A=B y A<B.
- Paridad par e impar.
- Generador de paridad.
- Comprobador de paridad.
- Aplicaciones en transmisión digital.
